Twist Solver

一个零构建的 3D 魔方跟解器

Twist Solver 把魔方状态、求解算法、3D 渲染和新手引导全部放在浏览器里运行。服务器只负责发送静态文件，真正的应用逻辑在前端完成。

What Runs Where

1

Cloudflare Pages 托管 HTML、CSS、JS、图片，并通过 CDN 发给用户。

2

Browser 加载 ES modules，创建 three.js 场景，并初始化魔方状态模型。

3

Solver 在本地浏览器线程里运行，输出一步步 move list。

4

UI Controller 把 move list 变成 glowing layer、黑色箭头、timeline 和 Next/Auto 控制。

No Backend Server

这是一个纯前端项目。它没有 Node/Express/API/database 之类的应用服务器。Cloudflare Pages 不是在运行你的业务代码，而是在静态托管文件。魔方状态、求解、校验、动画和交互全部发生在用户浏览器里。

Core Engine

  • CubeState 用整数坐标追踪 54 个 facelets，避免浮点漂移。
  • RubiksCube 负责 27 个 cubies、贴纸、动画队列和逻辑状态同步。
  • solver.js 提供 Beginner layer-by-layer 解法。
  • solverShort.js 把 CubeState 转成 cubejs facelet string，再调用 two-phase solver。

Deployment

当前生产环境是 Cloudflare Pages。每次 GitHub 的 main 分支更新，Cloudflare 会拉取 repo，按静态站配置发布到 twist-solver.pages.dev。

Brand Colors

UI 主色是 Twist Solver green，魔方颜色保持经典 WCA 风格，方便用户把屏幕提示对应到真实魔方。
